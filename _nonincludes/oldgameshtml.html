Tunnel Visioned

Unity Game / Android

Tunnel Visioned is a simple infinite runner, developed as a personal challenge to complete a game within 48 hours. Since initial completion in November 2015, it has been refined and released in January 2016.

Different

Unity Game / PC

Different was a group project for a school-based 72-hour game jam. A team of 7 students were tasked to develop a game with the following theme: Special, Fateful, Abstract.
